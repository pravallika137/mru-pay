# mru-pay
mru-pay is an educational fintech application developed as a demonstration project during college coursework in fintech application development. This project showcases a basic implementation of a digital payment system similar to PhonePe, with the public-facing branding displayed as "Mru Pay" while the internal codebase references it as phonepay.

**Disclaimer:** This is an educational PhonePe clone project named "mru-pay" (displayed as such), with internal code references to `phonepay`. Not affiliated with actual PhonePe.

# Technology stack
The application is built with a modern full-stack architecture:

Backend: Spring Boot (Java) - RESTful APIs for payment processing and transaction management

Frontend: React - User-friendly interface for seamless payment experiences

Database: MySQL - Reliable data storage for users, transactions, and payment history

API Testing: Swagger - Comprehensive API documentation and interactive testing of all endpoints

## Project Overview

This project demonstrates:
- User authentication and wallet management
- Money transfer between users
- Transaction history tracking
- RESTful API design for payment systems
- Full-stack development (backend + frontend)

**Note:** AI/chatbot assistance was used during development for learning and acceleration purposes.
